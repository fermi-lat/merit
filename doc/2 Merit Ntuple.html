Note: When viewed together, the Merit and Analysis ntuples are referred to as the Summary Ntuple. Together, these ntuples contain the entire event summary. The Summary Ntuple therefore represents the primary output of Gleam, the package that enables an end-user to process data in real or simulated mode.

Both Merit and Analyis ntuple data is organized in a tabular format with each row a fixed-length of data for a single event, and each column an ntuple variable. The contents of the these ntuples are described in the following tables.
  • Names in this ntuple are likely to change… underscores removed, consistent prefixes added, maybe "IM" changed to "Event"...

  • Text in BLUE = notes to the user.

IMvertexProbClassification tree probability that the vertex gives a better measure of the direction than the best track alone
IMcoreProbClassification tree probability that the event is in the core of the PSF
IMpsfErrPredClassification tree prediction of the PSF for this event, normalized to the 68% point predicted from an analytic model. That is, "1" means that this event is likely to have the nominal PSF; a lower number means that the PSF is likely to be better than nominal.
IMgammaProbClassification tree prediction of the probability that this event is a gamma rather than a background event.
